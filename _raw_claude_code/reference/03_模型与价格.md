# 03 模型与价格

## 来源

- https://docs.anthropic.com/en/docs/about-claude/pricing
- https://support.anthropic.com/en/articles/11940350-claude-code-model-configuration

## 关键摘录

### 官方价格表节选

- Claude Sonnet 4.5：Base Input Tokens 为 `$3 / MTok`，Output Tokens 为 `$15 / MTok`
- Claude Haiku 4.5：Base Input Tokens 为 `$1 / MTok`，Output Tokens 为 `$5 / MTok`
- Claude Opus 4.6：Base Input Tokens 为 `$5 / MTok`，Output Tokens 为 `$25 / MTok`
- 官方说明：“The 'Base Input Tokens' column shows standard input pricing, 'Cache Writes' and 'Cache Hits' are specific to prompt caching, and 'Output Tokens' shows output pricing.”

### Claude Code 切模型

- “The simplest way to change models is to use the /model command directly within Claude Code.”
- “Type /model and choose your desired model from the interactive menu.”
- “You can check your current model anytime by running /status in Claude Code.”

## 对本文可用的结论

- Haiku 明显便宜，适合翻译、改写、格式整理、提纲草拟一类低风险工作。
- Sonnet 更适合日常主力开发与分析。
- Opus 适合真正需要高推理强度、且结果价值能覆盖成本的任务。
- 把“模型选择”当成一个使用习惯，而不是默认一直开最强模型，才是真正的省成本。
