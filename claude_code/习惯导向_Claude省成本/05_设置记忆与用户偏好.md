# 习惯 5：设置记忆与用户偏好

## 原文定义

那些每次新对话都会重复交代的角色、风格和偏好，不要每轮都重讲，尽量保存到记忆与用户偏好里。

## 为什么原文这么强调

- 每次新对话都重新铺一遍背景，是重复消耗
- 角色、风格、偏好属于长期信息，不适合一直放在临时会话里
- 一旦沉淀下来，后续很多轮都不用再重建

## 场景要怎么理解

低效方式：

```text
会话 A 第 1 轮：我是一名营销人员，我喜欢轻松风格，偏好短段落
会话 B 第 1 轮：我是一名营销人员，我喜欢轻松风格，偏好短段落
会话 C 第 1 轮：我是一名营销人员，我喜欢轻松风格，偏好短段落
```

更符合原文意思的方式：

- 把角色、沟通风格、输出偏好存到记忆与用户设置
- 每次新会话直接在这个基础上开始工作

## 顺着原文补充的理解

这条习惯优化的是“跨会话重复背景”的成本。
