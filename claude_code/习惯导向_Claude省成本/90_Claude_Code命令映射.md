# Claude Code 命令映射

这页不是命令大全，而是把“省成本习惯”映射到 Claude Code 里真正有用的动作。

## `/model`

用法：切换当前会话模型。

适合习惯：

- 按任务切模型
- 先决定成本档位，再开始工作

## `/status`

用法：查看当前会话状态与模型。

适合习惯：

- 确认自己现在是不是还在高成本模型上

## `/memory`

用法：编辑或管理记忆文件。

适合习惯：

- 把长期背景沉淀为记忆
- 区分会话信息与长期记忆

## `/compact`

用法：压缩当前会话。

适合习惯：

- 长对话及时压缩
- 把噪音历史折叠为摘要

## `/init`

用法：帮助初始化项目级记忆和规则。

适合习惯：

- 把固定背景外置
- 给项目建立长期协作框架

## 实际建议

如果你是高频用户，可以把一条默认顺序记住：

```text
先 /model
再确认需不需要外部能力
长期规则放 /memory 或 CLAUDE.md
长对话用 /compact
项目开头先 /init 打底
```
