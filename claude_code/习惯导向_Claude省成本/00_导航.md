## 关于“轮次”的统一口径

- 一轮，就是一次发送
- 分三次发，就是三轮
- 一次发三个任务，仍然是一轮

所以原文里第 1、2、3 个习惯讲的重点分别是：

- 习惯 1：回答方向不对时，不要继续加新轮次纠错，优先编辑原消息
- 习惯 2：轮次已经很多时，要及时总结并开新对话
- 习惯 3：相关任务在第一轮就尽量合并，不要拆成多轮补发
